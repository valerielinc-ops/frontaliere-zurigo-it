Il Laboratorio di Intelligenza Visiva per i Trasporti ( VITA ) è alla ricerca di un ricercatore post-dottorato nell'area dell'IA generativa.

Gli interessi di ricerca di VITA si trovano all'intersezione tra Computer Vision, Machine Learning (Deep Learning) e Human-Robot Interaction per ridefinire il modo in cui le macchine interpretano le azioni umane, anticipano le intenzioni e interagiscono senza soluzione di continuità con le persone in vari contesti.
