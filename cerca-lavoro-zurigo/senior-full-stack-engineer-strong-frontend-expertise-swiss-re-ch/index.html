Requisiti principali

  • Il nostro team tecnico comprende professionisti con competenze multidisciplinari, che si trovano in vari paesi, offrendo soluzioni multiregionali altamente disponibili in tutto il mondo.
  • Forte esperienza di sviluppo dell'interfaccia utente con design reattivo, MUI, CSS-Modules, Tailwind CSS, Post CSS e Sass/Less.
  • Forte conoscenza di Redux Toolkit, Context API, Tan Stack Query, React Router/Tan Stack Router, route-driven UX, carico pigro, divisione codice e ottimizzazione delle prestazioni frontend.
  • Forte conoscenza dell'accessibilità web (WCAG), HTML semantico, ARIA, navigazione della tastiera, contrasto del colore e test di accessibilità automatizzati.
  • Forte esperienza di test con Jest, React Testing Library, user-event e l'esposizione a Cypress o Playwright.
  • Esperienza nella progettazione, nello sviluppo e nell'integrazione delle API, con la capacità di comprendere le implementazioni backend ed eseguire la risoluzione dei problemi end-to-end.
  • Forte conoscenza dei database relazionali (MSSQL, PostgreSQL), versione schema (Flyway), e ORM framework (Hibernate, JPA).
  • La progettazione, l'implementazione, il test e la fornitura di applicazioni cloud-native affidabili, ad alte prestazioni e scalabili utilizzando moderni servizi cloud e modelli architettonici.
  • Esperienza pratica con soluzioni di autenticazione e identità aziendali.
  • Esperienza con l'imballaggio delle applicazioni e la containerizzazione, tra cui la costruzione di utensili, Azure DevOps pipelines, Docker e Kubernetes.
  • Forte comprensione della progettazione sicura del software, delle pratiche di codifica sicure, dell'analisi del codice statico, della scansione della vulnerabilità e della bonifica.
  • Conoscenza di base di Terraform, inclusa la lettura delle configurazioni esistenti e l'attuazione di modifiche minori.
